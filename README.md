# Patient-Ventilator Asynchrony (PVA) Detection
This repository contains a framework to simulate mechanical ventilation waveforms and detect patient-ventilator asynchronies using the Equation of Motion.

## Features
- *Waveform Generator:* Physics-based simulation of lung pressure and flow.
- *PVA Injection:* Includes functions to simulate Double Triggering and Ineffective Triggering.
- *Detection Algorithm:* Slope-based analysis to identify asynchronous events.

## How to use
Click on the .ipynb file to view the code, or click the "Open in Colab" badge to run the simulation directly in your browser.
